# SOTM 2016 Website

The `gh-pages` branch is published under:

http://openstreetmap.github.io/stateofthemap-2016/

## Local installation

### Install Jekyll

See http://jekyllrb.com/docs/installation/

### Lauch local server

* `git clone git@github.com:osmbe/sotm2016.git`
* `cd sotm2016`
* `jekyll serve`
